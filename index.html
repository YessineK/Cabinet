⚙️ Configuration Google API

📋 Étapes pour obtenir votre Client ID :
1. Allez sur console.cloud.google.com
2. Créez un projet → APIs & Services
3. Activez Google Drive API et Google Picker API
4. Créez des identifiants → ID client OAuth 2.0
5. Type : Application Web
6. Origines autorisées : l'URL de votre app
7. Copiez le Client ID et API Key ci-dessous
